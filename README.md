# Usando o CodeSpaces

## O que é o CodeSpaces:
É uma plataforma de codificação na nuvem.

ATENÇÃO:
1. Para ter um CodeSpace é necessário ter um repositório em sua conta Github
2. Na conta gratuita é possível ter até 2 CodeSpaces ativos
